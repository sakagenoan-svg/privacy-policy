DevDroid AI プライバシーポリシー

最終更新日：2026年4月15日

SakageNoAn（以下「当社」）は、DevDroid AI（以下「本アプリ」）において、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーは、本アプリが収集する情報およびその使用方法について説明します。

1. 収集する情報

本アプリは以下の情報を収集します：

  • アカウント情報：GitHub / GitLab / Google OAuth を通じたメールアドレス・ユーザー名
  • リポジトリ情報：ユーザーが登録したGitHub / GitLabリポジトリの名前・構成
  • 利用状況：AI修正・ビルドの使用回数（プラン制限の管理目的）
  • FCMトークン：プッシュ通知の送信目的
  • 会話履歴：AI との会話内容（端末のローカルDBに保存）

2. 情報の使用目的

収集した情報は以下の目的で使用します：

  • 本アプリのサービス提供・機能の実現
  • プラン別機能制限の管理
  • ビルド完了・修正完了などのプッシュ通知の送信
  • サービスの改善・不具合の修正
  • お問い合わせへの対応

3. 第三者サービスへの情報提供

本アプリは以下の第三者サービスを利用しており、各サービスのプライバシーポリシーに従い情報が処理されます：

  • Anthropic Claude API：AIによるコード解析・修正（コード内容を送信）
  • GitHub / GitLab API：リポジトリへのアクセス・コードのpush
  • Supabase：認証・ユーザーデータの管理
  • Cloudflare Workers：APIゲートウェイとしての処理
  • Stripe：サブスクリプション決済の処理
  • Firebase（Google）：プッシュ通知・APK配布
  • ElevenLabs：音声合成（テキスト読み上げ）

4. データの保存・保護

  • 会話履歴は端末のローカルデータベースに保存され、サーバーには送信されません
  • APIキーなどの秘密情報はCloudflare Workers内でのみ管理され、アプリ側には保存されません
  • 通信はすべてHTTPS（TLS）で暗号化されています
  • サインアウト時に端末内のデータはクリアされます

5. データの保持期間

  • アカウント情報：退会まで保持
  • 利用状況データ：月次リセット（プラン制限カウンター）
  • 会話履歴：ユーザーがリセットするまで端末に保持

6. ユーザーの権利

ユーザーは以下の権利を有します：

  • 収集データの開示請求
  • データの訂正・削除の請求
  • アカウントの削除（アプリ内またはお問い合わせより）

7. 子どものプライバシー

本アプリは13歳未満の子どもを対象としておらず、意図的に13歳未満の個人情報を収集することはありません。

8. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されます。重要な変更がある場合はアプリ内またはメールにてお知らせします。

9. 準拠法

本ポリシーは日本法に準拠します。
