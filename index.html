スマホ 1 台で Android 開発

移動中も、通勤中も。
AI が書いて、ビルドする。

DevDroid AI は、スマートフォンから自然言語で指示するだけで Android アプリのコード修正・PR 作成・クラウドビルド・APK 配布までを完結させる SaaS です。

主な特徴

PC を開かずに、自然言語だけで Android アプリ開発が回ります。

AI によるコード修正

Claude AI に「ボタンの色を変えて」「バグを直して」と話すだけ。プロジェクト構造を理解した上で差分を生成します。

GitHub / GitLab 連携

OAuth で接続したリポジトリに対し、修正内容を自動 PR / MR 化。直接 push もユーザー設定で選択可能です。

クラウドビルド

GitHub Actions が APK をビルドし、Firebase App Distribution 経由で端末に配信。コマンドライン操作は不要です。

差分プレビュー

適用前に diff を確認できるため、AI の出力を吟味してから取り込めます。コード品質の検証機構も内蔵。

音声入出力

音声入力で指示、応答は自然な音声で読み上げ。手が塞がっている移動中もハンズフリーで開発できます。

チーム共有

リポジトリを owner / member / viewer ロールで共有。コードレビューやチーム開発もスマホで完結します。

エラー自動解決

ビルド失敗を検知して AI が原因解析と修正 PR を自動作成。深夜に発生したエラーも翌朝には対応済みに。

多言語対応

UI は日本語・英語に対応 (他 5 言語は順次拡充中)。AI との会話は日本語で最適化され、自然な対話を実現。

使い方

セットアップから最初の APK 配布まで、平均 5 分。

リポジトリ連携

GitHub または GitLab アカウントを OAuth で接続。対象リポジトリを選択します。

AI に指示

「ログイン画面のボタンを赤に」など、自然言語で修正依頼を入力します。

差分を確認 → PR

AI が生成した diff をプレビューし、確定すれば PR が自動作成されます。

ビルド & 配布

クラウドでビルドが走り、APK が Firebase 経由で端末にプッシュ通知されます。

よくある質問

本当にスマートフォンだけで開発できますか？

はい。コードの編集・差分確認・PR 作成・ビルド・APK 配布までクラウドで完結するため、PC を起動する必要はありません。複雑なリファクタリングや IDE 機能が必要な場合のみ PC をご利用ください。

対応しているプラットフォームは？

クライアントは Android アプリです。Google Play Store にて配布いたします。修正対象のコードベースは Kotlin / Java を中心に、Web 系言語にも対応しています。

ソースコードは外部に送信されますか？

AI に指示する際、修正対象ファイルの内容のみ Anthropic Claude API に送信されます。送信内容や API キーの管理方針はプライバシーポリシーをご確認ください。

無料プランから有料プランへの切替は？

アプリ内の設定画面から Stripe Checkout 経由でアップグレードできます。月単位のサブスクリプションで、いつでも解約可能です。

チーム共有で他のメンバーに何ができますか？

owner / member / viewer の 3 つのロールがあります。viewer はコード閲覧と AI への質問のみ、member はコード修正も可能、owner は招待・削除を含む管理権限を持ちます。

サービス開始直後の制約はありますか？

リリース初期は機能改善を優先するため、一部の高度な機能 (大規模リファクタリング等) で制限があります。順次機能拡張を予定しております。
